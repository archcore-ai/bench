# Бенчмарк токенов Archcore — кратко

[English](SUMMARY.md) · **Русский**

> Воспроизводимый замер: экономит ли Archcore токены для AI-агента при фиксированном качестве
> и против реалистичных бейзлайнов. Полная методология и данные: [`ANALYSIS.ru.md`](ANALYSIS.ru.md).

---

## Что мерили

Публичный Go-проект (`go-chi/chi`) с синтетической базой знаний проектных правил.
Каждая задача — вопрос, ответ на который это **произвольный токен, спрятанный внутри одного
документа**: угадать нельзя, можно только реально найти нужный документ. Пять способов доступа
сравнивались по **точным API-токенам** при **одинаковом качестве** (все дают верный ответ):

| Способ | Доступ к знаниям |
|--------|-----------------|
| **A — холодный** | нет доков; агент роется в исходниках |
| **B1 — preload** | все доки загружены в `CLAUDE.md` заранее |
| **B2 — файлы + индекс** | доки как файлы + курируемый `CLAUDE.md`-индекс |
| **B3 — файлы + греп** | доки как файлы, без индекса; агент сам ищет Grep'ом |
| **C — archcore** | доки как файлы; поиск + загрузка через Archcore MCP |
| **D — plugin** | как C, но с `--plugin-dir` и скиллом `/archcore:context` |

CLI-бенчмарк: 305 измерений (125 кроссовер + 180 воркнагрузка), 0 сбоев.
Plugin-бенчмарк: 120 измерений (3 trial × 5 доменов × 4 batch size × 2 arm).

## CLI: Archcore как MCP-сервер

Стоимость Archcore за задачу **не зависит от размера базы**. Preload в `CLAUDE.md` растёт
линейно. Разрыв расширяется с ростом базы без предела.

| Размер базы | CLAUDE.md preload | Файлы + греп | **Archcore CLI** |
|-------------|-------------------|--------------|------------------|
| 1 документ  | $0.102            | $0.103       | $0.127           |
| 20 доков    | $0.134            | $0.121       | $0.127           |
| 80 доков    | $0.238            | $0.123       | $0.127           |
| 320 доков   | $0.681            | $0.112       | $0.127           |

- Кроссовер с `CLAUDE.md` preload: **~27 документов**. Дальше Archcore дешевле.
- При 320 доках: Archcore **в 4.6 раза дешевле** preload ($0.127 против $0.681).
- Против «файлы + греп»: **паритет** — оба $0.11–0.13/задачу по всему диапазону.
- Качество: 100% правильных ответов на всех размерах базы.

Механизм: preload несёт весь KB в контексте при каждом ходу (растёт линейно); Archcore
загружает только нужный срез (плоско). Стоимость = 3 хода × ~$0.04/ход.

## Plugin: Archcore + интеграция с Claude Code

Plugin добавляет `/archcore:context <область>` в сессии Claude Code. Когда в одной сессии
задаётся несколько вопросов об одной области, фиксированный overhead (схемы MCP, system prompt,
контекст исходников) платится один раз и распределяется по всем вопросам.

**Стоимость за вопрос при разных размерах батча** (медиана, 5 доменов, 3 trial):

| Вопросов / сессия | Plugin (D-arm) | Сырой MCP (C-arm) | vs. отдельные CLI-вызовы |
|-------------------|---------------|--------------------|--------------------------|
| 1                 | $0.163        | $0.150             | −29%*                    |
| 2                 | $0.075        | $0.067             | −41%                     |
| **4**             | **$0.032**    | **$0.033**         | **−74%**                 |
| 8                 | $0.015        | $0.014             | −88%                     |

*Бейзлайн — CLI-отдельные вызовы по $0.127/вопрос.

**Полная воркнагрузка** — 20 задач (5 доменов × 4 вопроса на домен):

| Стратегия | Итого | Экономия |
|-----------|-------|----------|
| 20 × CLI отдельно | $2.54 | — |
| Plugin батч (5 × 4 вопроса) | **$0.649** | **−74%** |
| Сырой MCP батч (5 × 4 вопроса) | $0.722 | −72% |

Plugin на **10% дешевле** сырого MCP-батча на полной воркнагрузке. Для доменов `errors` и
`testing` (ADR-документы) Plugin дешевле сырого MCP на 20% при батче 4 — скилл контекста
помогает агенту точнее находить нужные документы.

## CLI vs Plugin

|  | CLI (один вызов/вопрос) | Plugin (батч-сессия) |
|--|------------------------|----------------------|
| Стоимость/вопрос | $0.127 | $0.163 (N=1) → $0.032 (N=4) |
| Безубыточность vs CLI | — | N ≥ 2 вопросов на область |
| Полная воркнагрузка 20 задач | $2.54 | $0.649 (−74%) |
| Стабильность стоимости (IQR при N=4) | $0.008/q | $0.001/q |
| Лучше для | Скрипты, одиночные запросы | Интерактивные сессии |
| vs CLAUDE.md preload | 4.6× дешевле при 320 доках | — |

## Итог

**CLI** подходит для запросов к базе знаний из скриптов, одноразовых пайплайнов или там, где
сессия содержит один вопрос. Стоимость фиксирована на $0.127/задачу вне зависимости от размера
базы, на паритете с «файлы + греп» — без обслуживания индекса.

**Plugin** подходит для интерактивных сессий Claude Code, где разработчик задаёт несколько
вопросов об одной области. При 4+ вопросах в сессии стоимость падает до $0.032/q (−74%).
Точка безубыточности — ~2 вопроса.

**Что нельзя заявлять:** Archcore не дешевле любого бейзлайна. Против «файлы + греп» — паритет,
не выигрыш. Честный токен-нарратив: *масштабируемость против preload* и *паритет с грепом*.
